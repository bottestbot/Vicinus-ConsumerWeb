## üîé Smart Search ‚Äî natural-language ‚Üí DDF filters (2026-07-22)
_A single plain-language field on the dashboard ("house with a pool near a park and school, under $1.4M") that resolves to editable filter chips and runs the existing DDF search. Ships behind a Smart/Classic toggle (the toggle is the feature flag's UI face)._

_**Architecture ‚Äî the one rule that governs every task below: the LLM parses the sentence, it never touches a listing.** The model sees only the user's text and emits a validated JSON filter object; DDF is then queried deterministically by the existing `SearchService`. Nothing from CREA enters a prompt, nothing is LLM-generated about a listing, so this is **DDF-compliant by construction** ‚Äî the same posture as the IQ Brief, inverted (there the LLM writes prose over facts; here it reads text and emits filters)._

_**The four buckets every parsed token sorts into** ‚Äî the whole plan is built around them: **A Native** (maps straight to `SearchQueryDto` + a `Property` column ‚Äî ships day 1, no new infra), **B Amenity** (true of the listing but buried in `Property.exteriorFeatures` JSON ‚Äî needs SMART-05), **C Proximity** ("near a park / walk to a school" ‚Äî the differentiator, backed by the owned `NeighbourhoodPoi`/OSM layer, needs SMART-06/07), **D Degrade** (can't honestly answer ‚Äî rentals, "quiet street", "good schools" ‚Äî refused in plain language, never faked)._

_**Schema truths this is built on (verified 2026-07-22):** (1) there is **no `StructureType` column** ‚Äî `Property.propertySubType` is the only stored dwelling-form field despite the DTO exposing `structureType`, so house/condo/townhouse needs an explicit normalization map. (2) `exteriorFeatures` is `Json?` ‚Äî not queryable for `pool`/`garage` without SMART-05. (3) `NeighbourhoodPoi` stores `lat`/`lng` floats scoped **by neighbourhood** with no PostGIS + no spatial index, so arbitrary point-radius ("within 800m of this listing") needs the POIs promoted to a listing-queryable spatial layer. (4) per the DDF/REAW tier there are **no residential rentals** in the feed ‚Äî rental intent must hard-degrade._

_**Reuse the VERTEX shared Gemini provider** (VERTEX-03) rather than a fourth call site; if VERTEX hasn't landed, write the parser against `@google/genai` so it ports for free._
